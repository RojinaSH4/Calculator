# Rojina Calculator

A simple **Windows Forms Calculator** built in C# with basic arithmetic operations and additional functions.

## Features

- Basic arithmetic: `+`, `-`, `*`, `/`
- Advanced operations: 
  - Square (`x²`) and Cube (`x³`)
  - Square root (√)
  - Inverse (`1/x`)
  - Factorial (`!`)
  - Pi constant (`π`)
- Memory of previous input and operations
- User-friendly interface with **color transition effects** and sliding panel
- Error handling (e.g., division by zero, factorial for non-integers)
